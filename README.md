# UniHub — Your Academic Command Centre

UniHub is a beautifully designed, local-first web platform that brings together all the tools modern students need to manage their academic journey. Since it is client-side only, all data is securely saved directly in the browser's `localStorage` — no account creation, server, or cloud storage required.

---

## 🚀 Key Features

*   **🍅 Pomodoro Focus Timer & Task Manager**
    *   Customizable study intervals with automatic short/long break schedules.
    *   Built-in task manager: assign tasks to specific subjects, set priority tags, and track progress.
    *   Gamification elements: gain Focus XP and maintain a daily study streak.
*   **📊 Interactive Dashboard**
    *   Visualize your overall progress, GPA trends, focus sessions, and task completion rates.
    *   Clean graphs and cards displaying real-time statistics.
*   **🧮 GPA Calculator**
    *   Calculates semester-specific and cumulative GPA over multiple levels/semesters.
    *   Features a pre-configured curriculum (IAT) or allows uploading custom curriculums via standard JSON files.
    *   Supports adding custom individual course rows dynamically.
*   **🎵 Focus Music & Synthesized Ambience**
    *   Upload local music playlists to play directly inside the browser.
    *   Built-in synthesizer playing ambient tracks like *Low Rain*, *Alpha Beats*, and *Tide Wave*.
*   **📚 Study Resources Organizer**
    *   Keep track of academic URLs, reading links, and lecture bookmarks organized in one clean interface.
*   **🌓 Sleek Dark/Light Theme Support**
    *   Fully integrated styling system with dynamic themes, smooth glassmorphic UI, and rich animations.

---

## 🛠️ Technology Stack

*   **Structure**: Semantic HTML5
*   **Styling**: Vanilla CSS3 (custom CSS variables, responsive grids, flexbox, glassmorphic filters, keyframe animations)
*   **Logic**: Pure client-side JavaScript (Vanilla JS, Web Audio API, LocalStorage API)
*   **Icons**: FontAwesome 6.5.0 (CDN)

---

## 🏃 Getting Started

Because UniHub runs entirely in your web browser, there is no setup required:

1. Clone or download this directory.
2. Double-click the [index.html](index.html) file to open it in your default web browser (Chrome, Firefox, Safari, Edge, etc.).
3. Add subjects, log your focus sessions, and start tracking your GPA!
